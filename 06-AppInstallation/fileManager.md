# Thunar

- Arch

```bash
sudo pacman -S thunar tumbler ffmpegthumbnailer tumbler-extra-thumbnailers
```

- Debian

```bash
sudo apt install tumbler tumbler-plugins-extra ffmpegthumbnailer
```

# Dolphin

- Arch

```bash
sudo pacman -S dolphin
```

- Debian

```bash
sudo apt install dolphin
```
